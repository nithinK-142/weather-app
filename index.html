Weather App

This is a simple weather application that displays the current time, 
date, and weather information for a specified city.

   Usage
-> Enter the name of a city in the input field.

-> Click the "Enter" button to retrieve the weather info.

-> The weather information such as temperature, humidity, wind speed
   and description will be displayed.

-> An icon representing the weather condition will also be displayed.

-> To clear the search and reset the display, click "Clear" button.
